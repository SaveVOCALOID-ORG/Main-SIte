什么是 SaveVOCALOID.ORG?


SaveVOCALOID.ORG 是一个由 VOCALOID 社区发起的项目。

最近，在国内的网站和平台上，VOCALOID 作品大量地被删除或隐藏。此外，VOCALOID 圈子也出现了过于低龄化等令人担忧的现象。

我们希望能够尽一份微薄之力，应对 VOCALOID 社区所面临的问题。例如：对 VOCALOID 作品进行存档，以免 P 主和爱好者们的心血毁于网络平台不合理的政策；对 VOCALOID 文化进行适当的科普，以免新人被错误的信息误导。

要了解更多信息，请→阅读我们的文档。


我如何帮助你们？


我们接受多种形式的帮助。

您可以帮忙撰写文档条目，它的 Github 仓库在 SaveVOCALOID-ORG/Docs。

如果您不熟悉 Git 的操作，您可以直接将内容投稿到我们的 Bilibili 账号私信或是我们的电子邮箱。

或者——把这个站点转发给您的朋友们，这已经足够了。


捐赠
